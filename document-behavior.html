Recommendation for WIF Japan

Swipe Actions Quick

Best for power users who need fast access. Swipe left → Edit/Delete. Swipe right → Share. Add a subtle hint on first use.

Tap → Detail View Primary

Tap the row to open detail view with full information. Share & Edit buttons visible in action bar. Delete in ⋮ menu.

Delete Confirmation Required

Always show confirmation dialog: "Delete Invoice WIF250170? This cannot be undone." with Cancel and Delete buttons.

Recommended Combination

Use Pattern 1 (Swipe) + Pattern 3 (Detail View):

• Tap → Opens detail view with Share & Edit in action bar
• Swipe left → Reveals Edit + Delete (quick access)
• Swipe right → Reveals Share
• Long press → Optional: iOS context menu as fallback
• Delete always → Requires confirmation dialog
